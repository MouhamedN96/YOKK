# YOKK

The platform for African builders.

Learn. Build. Grow. Ship. With AI that understands your context and a community that has your back - even when your network doesn't.

## What is YOKK?

YOKK is where Africa's builders connect, learn, and grow. Developers, entrepreneurs, marketers, traders - anyone building something that matters.

### Features

- **Community** - Share knowledge, ask questions, find your people
- **Launches** - Showcase your product, service, or hustle
- **Bo AI** - Your assistant that gets the African context
- **Voice Comments** - Talk it out when typing isn't enough
- **Works Offline** - Because your grind doesn't stop when the network does
- **Level Up** - Earn XP, grow your reputation, unlock achievements

## Development

```bash
npm install
cp .env.example .env.local
npm run dev
```
